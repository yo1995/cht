Coding.net mirror for the blog

2016-08-28
html
Git, Jekyll, html

设定ssh公钥

截止日期2017.1，毕业前及时更换所有注册邮箱。

coding-pages设置相关

默认分支为coding-pages，默认域名为{username}.coding.me，可绑定域名。

访问速度

更换了以后确实访问速度提升很多，但制约速度的主要因素还是一些文件的载入。尤其是js。

其它

之前尝试采用git到多个分支的方法：
更改.git目录下config文件，增加pushurl = git@git.coding.net:{username}/{username}.git即可推送至多个仓库同一分支。
然而因为coding默认分支与github不相同，因而无法使用此方法。

同时，由于不同页面的config.yml亦存在一些差异，因而放弃同目录多远程仓库推送，改为双目录。
准备以后coding更新速率低于github，每周同步一次这样。
